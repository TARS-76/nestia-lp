## 2. カラーパレット（v02確定 — 相手のナチュラル軸 + 差し色）

> 方針: 相手の世界観（生成り・砂・布・植物）を**地**にして、差し色のくすみピンク + ピンクゴールドを**点**で効かせる。
> 「色を変えた」ではなく「あなたの世界観を少しだけ洗練させた」という提案に見せる。差し色の根拠は実ロゴの淡いピンク円アイコン。

```css
/* 4-6色の名前付きパレット */
--color-bg:        #F4EFE9;  /* 生成り（内装・公式ビジュアル準拠・背景主） */
--color-surface:   #FFFFFF;  /* カード面 */
--color-text-main: #6E5F50;  /* ロゴの茶（文字主） */
--color-text-sub:  #9C8E7E;  /* くすみベージュ（サブ文字） */
--color-primary:   #C9A8A0;  /* くすみピンク（差し色・ロゴ円アイコン由来） */
--color-accent:    #C2A878;  /* ピンクゴールド寄り真鍮（CTA・強調） */
```

**使用イメージ**

- CTA（HPB予約）: `--color-accent`（ピンクゴールド）
- セクション区切り・eyebrow・小装飾: `--color-primary`（くすみピンク）
- 価格・強調テキスト: `--color-accent`
- 背景: `--color-bg` を基調に、カードは `--color-surface`

`[施術後]` 実ロゴの正確な色を採取して `--color-primary` を微調整する。
