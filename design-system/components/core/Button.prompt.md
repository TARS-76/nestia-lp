Pill-shaped primary action button — use for any CTA or form submit; plum by default.

```jsx
<Button>ご予約はこちら</Button>
<Button variant="accent" size="lg">本日空きあり</Button>
<Button variant="outline">メニューを見る</Button>
```

Variants: `primary` (plum, default) · `accent` (petal rouge — reserve for one emphatic CTA) · `outline` · `ghost`. Sizes `sm | md | lg`. Pass `block` for full-width, `iconLeft` / `iconRight` for icons, `as="a"` for a link.
